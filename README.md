### 🎯 What I'm Learning Next

**Web Development track**
- [ ] JavaScript fundamentals deeper dive (DOM manipulation, async/await, fetch API)
- [ ] A frontend framework — **React** is a strong next step
- [ ] Basic backend concepts — **Node.js + Express**
- [ ] Git branching, pull requests, and collaborative workflows
- [ ] Responsive design & accessibility basics

**Cybersecurity track (beginner-friendly entry points)**
- [ ] **Networking fundamentals** — TCP/IP, DNS, HTTP/HTTPS
- [ ] **Linux basics** — command line comfort is a prerequisite for almost everything in security
- [ ] **OWASP Top 10** — the most common web vulnerabilities (SQL injection, XSS, CSRF, etc.), pairs naturally with web dev knowledge
- [ ] Python basics — most security tooling and scripting leans on Python

> 💡 Tip: Web dev + cybersecurity is a great combo — understanding how sites are *built* makes it much easier to understand how they can be *attacked and defended*. Consider intentionally building small vulnerable practice apps (safely, locally) to see security concepts in action.
